# 몬티홀 실험실

몬티홀 문제를 직접 플레이하고 자동 시뮬레이션으로 검증할 수 있는 한국어 웹앱입니다.

## 기능

- 세 개의 문 중 하나를 고르는 대화형 게임
- 선택 유지/변경 전략별 개인 승률 기록
- 100회부터 100,000회까지 자동 반복 실험
- 모바일과 데스크톱에 대응하는 반응형 화면
- 별도 설치나 빌드 과정 없이 실행

## 실행

`index.html`을 브라우저에서 열면 됩니다.

로컬 서버를 사용하려면:

```bash
python -m http.server 8000
```

그다음 <http://localhost:8000>에 접속하세요.

## 핵심 원리

처음 고른 문에 자동차가 있을 확률은 1/3입니다. 사회자가 염소가 있는 문을 열고 나면, 선택을 바꾸는 전략은 나머지 2/3의 확률을 가져오므로 장기적으로 약 두 배 높은 승률을 보입니다.
