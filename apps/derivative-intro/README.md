# 곡선 속에 숨은 직선

미분을 처음 도입하는 수업에서 사용할 수 있는 동기유발용 웹 활동입니다.

## 수업 흐름

1. 곡선 위 점 P를 움직이며 위치에 따라 굽은 정도가 달라짐을 관찰합니다.
2. 큰·중간·작은 돋보기로 바꾸며 관찰 범위 `Δx`를 줄입니다.
3. 작은 구간에서 곡선이 직선처럼 보이는 현상을 말로 설명합니다.
4. `접선 발견하기`를 눌러 할선이 접선으로 가까워지는 아이디어를 확인합니다.
5. “한없이 가까워진다를 수학으로 어떻게 표현할까?”라는 질문으로 극한과 미분 학습을 연결합니다.

별도 설치 없이 `index.html`을 열어 실행할 수 있습니다.
